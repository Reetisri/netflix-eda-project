# Netflix EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset.
The goal is to clean data, analyze patterns, and visualize insights.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Most movies were added after 2015
- Top genres, countries, directors
- Distribution of release years
- Ratings breakdown

## How to Run
1. Download dataset from Kaggle
2. Run Netflix_EDA.ipynb
